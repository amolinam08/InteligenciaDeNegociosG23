Esta aplicación permite detectar intención de las reseñas de las peliculas a partir de textos, mensajes o publicaciones escritas por usuarios. El trabajo de predicción se realiza a partir de un modelo de clasificación entrenado por medio de un algoritmo Random Forest. El modelo fue entrenado con cerca de 5000 reseñas. A continuación, escoja si quiere obtener una predicción a partir de un texto o si quiere acceder al API del modelo. Desde el API, podrá obtener predicciones de multiples datos y a su vez reentrenar el modelo.
